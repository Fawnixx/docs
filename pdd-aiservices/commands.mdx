## Player Commands

| Command | Default Keybind | Description |
|---|---|---|
| `/aiservices` | `F6` | Open the AI Services tablet |

<Info>
  The command name is configurable via the in-game admin panel. The `F6` keybind can be disabled by setting `Config.EnableKeybind = false` in `config.lua`.
</Info>

---

## Admin Commands

| Command | Description |
|---|---|
| `/serviceadmin` | Open the tablet directly to the admin panel |

Admin commands are restricted to players with a license listed in `Config.AdminLicenses`.
