# PDD Advanced Loyalty

**Version:** 3.1.0 | **Framework:** QBCore | **Author:** PhatDaddy Development LLC

PDD Advanced Loyalty is a comprehensive player engagement and monetization system for QBCore FiveM servers. It combines multiple reward systems into a single, fully-integrated resource.

## Features

<CardGroup cols={2}>
  <Card title="VIP Shop" icon="crown">
    Coin-based VIP reward store with purchasable perks, vehicles, items, and more.
  </Card>
  <Card title="Playtime Rewards" icon="clock">
    Automatic rewards for players based on total server playtime milestones.
  </Card>
  <Card title="Battlepass" icon="trophy">
    Seasonal free and premium tracks with XP-based progression and milestone rewards.
  </Card>
  <Card title="Deal of the Day" icon="bolt">
    4-slot rotating daily deals with admin-managed pool and automatic expiration.
  </Card>
  <Card title="Coin Packages" icon="coins">
    Tebex-integrated purchasable coin bundles for the VIP shop.
  </Card>
  <Card title="Referral System" icon="users">
    Player referral codes with starter packs, milestone bonuses, and anti-abuse protection.
  </Card>
  <Card title="Promo Codes" icon="tag">
    Admin-created promotional codes redeemable for coins, items, or VIP time.
  </Card>
  <Card title="Multi-Language" icon="globe">
    Full locale support across 10 languages.
  </Card>
</CardGroup>
